--- layout: post title: Sumber Terbuka Atau Tertutup? tags: [diary] --- Mana jagoan saya diantara sumber terbuka atau tertutup jika saya harus memilih satu dari keduanya? Kalau menurut saya keduanya bagus, ada kelebihannya juga ada kekurangannya dan tentunya saya sudah menggunakan hampir dari keseluruhannya, jadi nanti saya akan jelaskan secara rinci setelah yang satu ini, saya bukanlah fanboy atau orang yang membela garis keras terhadap suatu produk karena pasti ada cacatnya juga dibalik yang positifnya, jadi harus imbang tidak boleh berat sebelah, kebanyakkan orang mungkin lebih menyukai pada hal yang positifnya saja tapi yang cacatnya ini menjadi bagian penting untuk tidak dilupakan, sumber terbuka maupun sumber tertutup seakan menjadi suatu agama baru dalam dunia teknologi tapi jika kita mengkombinasikan keduanya pasti bagus daripada menggunakan salah satunya saja dan menolak untuk menggunakan suatu teknologi baru, teknologi digunakan untuk mempermudah bukan mempersulit

Saya dari dulu kebanyakkannya pakai sumber tertutup, handphonenya pakai sistem operasi java dan untuk komputernya menggunakan windows, disaat yang lain sewaktu Sekolah Menengah Pertama udah pakai android saya masih pakai sistem operasi java dan saya juga pernah menggunakan symbian juga, symbian ini sistem operasi terbuka sebenarnya karena bisa dijalankan dalam berbagai macam perangkat tapi tetap saya pilihan saya jatuh kepada java, bahkan sampai selesai Sekolah Menengah Atas pun saya masih setia menggunakan sistem operasi java ini dibandingkan dengan android karena kalau menurut saya bahwa java ini mirip dengan iOS di masa lampau, apalagi kalau pakai sistem operasi javanya di handphone sony ericsson, pasti anda pun sudah paham perbedaannya, begitu cepat, tidak lag, aplikasi ukuran kecil tapi luar biasa fungsinya, bisa multitasking beberapa aplikasi dan irit kuota internetnya, saya pun masih pakai facebook dan google maps sewaktu menemani saya berkeliling di kota jakarta untuk menunjukkan lokasi saya dimana dan menunjukkan tempat wawancara pekerjaan sewaktu selesai Sekolah Menengah Atas, jadi dari sinilah bisa ditarik kesimpulan bahwa performa memang lebih bagus di sumber tertutup daripada sumber terbuka, namun untuk masalah kelemahannya adalah apabila sudah bubar pengembang dari aplikasinya atau sudah tidak memutuskan untuk dukungannya maka bubarlah sistem tertutup ini, performa bagus tapi untuk jangka panjang aplikasi rasanya kurang begitu tahan lama dan tidak mendapatkan kebebasan untuk memilih, sudah ditentukan dan bug sepenuhnya diperbaiki sama pihak pengembangnya jadi saya selaku pengguna tidak bisa melakukan apapun lagi untuk memperbaikinya

Saya juga pakai iOS pun enak dan lancar, mulai dari iPhone generasi 3 sampai 5 sudah pernah saya pakai namun setelahnya saya berhenti untuk menggunakannya dikarenakan kejar-kejaran dukungan aplikasi dan saya pun tak bebas selaku SuperUser atau pengguna yang paham tentang dalamnya sistem operasi mau dioprek pun kurang maksimal, akhirnya saya menggunakan android lagi setelah sekian lama nyaman pada ekosistem sumber tertutup ini, saya menyukai android karena tidak ada yang bisa menyembunyikan apapun dari pandangan saya, semuanya transparan dan jelas, untuk masalah keamanan sistem operasi bisa saya sesuaikan dengan kebutuhan jadi sangat maksimal namun untuk performa sudah kalah telak, android tak akan pernah sehebat iOS kecuali pengembang aplikasinya memperkecil jadi ukuran Lite agar bisa melibas iOS karena android dahulu kala tujuannya untuk melawan blackberry bukan iOS pada iPhone jadinya masih keteteran, tapi saat ini dan di masa depan saya akan tetap menggunakan sistem operasi terbuka, untuk handphonenya saya masih setia pada android dan untuk di komputernya saya tetap menggunakan linux karena saya yakin sumber tertutup ini tidak selamanya dan pasti jatuh juga.
