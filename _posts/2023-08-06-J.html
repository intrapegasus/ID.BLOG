--- layout: post title: Bagaimana Saya Membuat Musik? tags: [diary] --- Jika sebelumnya saya sudah membagikan beberapa cuplikan lagu yang ada notasi dan liriknya tersebut maka saya mencoba untuk menyegarkan kembali pikiran saya dengan pembuatan musik ini, takutnya saya lupa makanya ditulis disini sekaligus menjadi kenang-kenangan bagi saya dan para pembaca blog saya, saya pada awalnya membuat musik ini didasarkan karena sewaktu Sekolah Menengah Pertama saya sudah mempunyai musik sendiri yaitu lagu "Merah Putih" yang diciptakan dari pianika, maka sewaktu Sekolah Menengah Atas sampai setelahnya saya berfikir untuk menyempurnakan lagi kemampuan saya dalam musik, bukan hanya sekedar mendengarkan lagu saja tapi menjadi seorang musisi yang amatiran atau tidak ada latar belakang mengenai musik tapi tidak ada salahnya jika saya mencoba, kalau tidak mencoba maka saya tidak akan pernah tahu apa hasilnya, hasilnya adalah gagal jika saya tidak mencoba.

Saya membuat musik itu langsung cari di YouTube waktu itu saya lagi ada di warnet, karena saya suka ke warnet untuk browsing sesuatu hal sembari melihat lowongan pekerjaan dari jobstreet dan jobsdb, ada sebuah tutorial bagaimana caranya membuat musik dari sebuah aplikasi yang bernama FLStudio, setelah saya tonton maka saya download yang trialnya kalau tidak salah karena saya ingat bahwa FLStudio ini aplikasi berbayar dan setelah saya download dan saya amati beberapa instrumennya saya kurang cocok dengan aplikasi yang satu ini dikarenakan terlalu banyak atau terlalu rumit untuk saya pahami, akhirnya saya mencari alternatif lainnya, karena FLStudio tersebut hanya untuk Windows saja berhubung saya pakai Linux maka sudah barang tentu saya tidak dapat memasangnya lalu saya ketemu satu aplikasi yang serupa tapi tak sama dengan FLStudio, aplikasinya bernama LMMS, Kenapa sih saya download ini? Bukan karena ingin buat musik pada awalnya melainkan mengikuti sebuah kompetisi BimaDay dari Tri kalau tidak salah, waktu itu pemenangnya dapat hadiah dan salah satunya yaitu pergi ke luar negeri di Amerika Serikat sana, saya ikutan dan sampai malam saya menunggu ternyata bukan saya pemenangnya, tapi saya tidak menyerah sih, saya akhirnya mencoba lagi dan lagi sampai menjadi bagus.

Setelah kalah kompetisi, saya membuat musik lagi dan mencoba untuk kesekian kalinya, setelah jadi saya tawarkan kepada label musik dan pada akhirnya ditolak, lalu saya tidak menyerah maka saya buat sesuatu yang bagus dan saya pasarkan lagi pasar musik gratis lisensi, jadi saya buat musik tapi tidak komersial, musik sumber terbuka dan saya mengunggahnya ke Jamendo waktu itu, tidak disangka banyak orang menyukainya dan lagu saya pernah masuk trending salah satu lagu atau album kalau boleh saya bilang, sungguh luar biasa sebenarnya karena berarti lagu saya tidak jelek-jelek banget lah, lagunya masih seputaran instrumental klasik, setelahnya saya buat genre yang lainnya seperti rock, dan untuk popnya saya buat di iPhone saya menggunakan Medly, lagunya juga bagus tapi memang sengaja gak saya pasarkan sih, setelah dibuat lalu diunggah ke YouTube lalu saya hapus permanen dan saya tidak memiliki arsipnya lagi, saya sudah membuat lagu cukup banyak dan mungkin kedepannya saya akan mencoba lagi untuk memasarkan musik secara komersial dan bukan sumber terbuka seperti sebelumnya jika ada waktu, dana dan kemauan serta infrastruktur penunjang lainnya, satu lagi yang tak kalah pentingnya lagi yaitu ide, tanpa ada ide maka musik tidak akan tercipta, jadi terimakasih banyak untuk kalian yang pernah suka atau pernah tahu lagu saya dan saat ini memang saya belum ada rencana atau belum lagi buat lagu, saya menjadi penikmat lagu saja dahulu sembari memikirkan strategi yang pas, atau saya latihan hip hop atau rap mungkin keren, melatih lidah juga beberapa beat pada musiknya.
